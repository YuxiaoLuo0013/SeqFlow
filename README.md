# SeqFlow

This repository contains the official code for our paper:

**From Graph to Sequence: Distance-Aware Autoregressive Modeling for Dynamic Origin-Destination Flow Generation**

SeqFlow models dynamic origin-destination flow generation by transforming graph-structured OD dynamics into a distance-aware autoregressive sequence modeling problem.

## Datasets

The U.S. datasets used in this work can be obtained from [tsinghua-fib-lab/CommutingODGen-Dataset](https://github.com/tsinghua-fib-lab/CommutingODGen-Dataset).

The China datasets will be released later.

## Note

This repository currently provides the core code for the main experiment. Datasets and generated experiment outputs are not included.
